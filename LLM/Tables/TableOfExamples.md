## How to maintain this list

When adding a new XML example to the repository:

1. Place the XML file in the appropriate subfolder under `Objects/` or `Frames/`.
2. Add an entry to this table in the correct alphabetical position.
3. If the object already has one example (a flat bullet), convert it to a `<details>` collapsible section and add both old and new entries inside.
4. If the object already has a `<details>` section, add the new entry inside and update the example count in the `<summary>` tag.
5. Use the relative path format `../../Objects/<ObjectName>/<Filename>.xml` or `../../Frames/<FrameName>/<Filename>.xml`.
6. Include a short description after the link (e.g., "Minimum profile example", "Nordic Profile example").
